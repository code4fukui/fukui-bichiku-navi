# 備蓄ナビえちぜん

越前市防災備蓄ナビゲーションサイトです。自分に合った備蓄品を検索できたり、備蓄品カードを印刷して学習することができます。

## デモ
https://bichikunavi.code4fukui.org/

## 機能
- 自分の家族の人数や年代別に必要な備蓄品を提案
- 備蓄品カードの印刷・学習
- 災害対策ゲーム「サイガイクン」を遊べる
- 越前市の避難所マップを閲覧できる

## 必要環境
特になし

## 使い方
サイトにアクセスし、必要に応じてページ間を移動して情報を確認・活用できます。

## データ・API
- 備蓄品リスト: https://bichikunavi.code4fukui.org/opendata/list.html
- 備蓄カード: https://bichikunavi.code4fukui.org/opendata/card.html
